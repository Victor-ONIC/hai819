# hai819
Projet clone Minecraft pour l'UE Moteur de jeux (HAI819I)

## Install dependencies
```
sudo apt install \
    libglfw3-dev \
    libglew-dev \
    libglm-dev \
    libx11-dev \
    libxi-dev \
    libxrandr-dev \
    libxinerama-dev \
    libxcursor-dev
```
## Build and run
```
mkdir build && cd build
cmake ..
make && ./mc
```
